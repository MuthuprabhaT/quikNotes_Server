# Quik Note (Notes App) - Backend  

This is the backend of the Notes App, built using Node.js, Express.js, and MongoDB. It handles all server-side logic, including API endpoints for creating, deleting, and updating notes.  

## Features  

- RESTful APIs for CRUD operations on notes  
- Database integration with MongoDB  
- Middleware for error handling and request validation  

## Technologies Used  

- **Node.js**  
- **Express.js**  
- **MongoDB**
